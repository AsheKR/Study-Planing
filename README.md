### 제목 

`Type: 제목`

Type은 아래의 종류가 있다.

```text
Feat: 새로운 기능을 추가할 경우
Fix: 버그를 고친 경우
Docs: 문서를 수정한 경우
Style: 코드 포맷 변경, 세미콜론 누락, 코드 수정이 없는 경우
Refactor: 프로덕션 코드 리팩터링
Test: 테스트 추가, 테스트 리팩터링 (프로덕션 코드 변경 없음)
Chore: 빌드 태스트 업데이트, 패키지 매니저를 설정할 경우 (프로덕션 코드 변경 없음)
```
